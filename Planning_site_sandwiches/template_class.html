Brief

This is a Sandwich ordering program with which the user can select from multiple options of different functions. These include printing out the sandwich menu with prices, printing out the order menu (the items the user has added to their order), adding an item to their order along with the quantity, editing the order list, this editing include the changing of quantity, deleting an item as a whole, or canceling this function incase it was a mistake. More options include, deleting the order list and starting a new order and entering the customers details which include a dilivery or pickup option, getting the users name and phone number and the dilivery address if they chose the dilivery option. The finial function they can choose is confirming the order - this confirms the order list and the customer details. Finally, there is a quit option incase the user wishes to quit the program.

In Sprint Two, as stated in Sprint One reflection, I added an order list and an adding to order list function. Adding the order list (order_list = []) was relatively simple, it was the printing of the order list and some parts of the adding to order list function that I struggled with. If the user did not have anything in their order list I had to make sure that they knew that, which was easy enough. However, when they did have some items in their order list I struggled a little bit with how to print it out to them because I used dictionary. After I figured out how to do that, with some research and help from others, the next problem I had to tackle was that of figuring out how to multiply the original price of a sandwich by the quantity the user wished to add to their order. I figured out, with some help, that I had to create total_amount += item['total_price'] and then slightly later in the function, I printed out: print(f"Total: ${total_amount:.2f}"). Those were the struggles I faced with the reveiw_order function. The adding_item to order function was slightly more challenging. From here I will create a start new order function - this will delet everything that the user had added to their order, so they can start again fresh. I will also implement and edit order function.
